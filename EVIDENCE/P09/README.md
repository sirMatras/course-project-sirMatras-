## P09 — SBOM и SCA артефакты

В эту директорию CI складывает результаты проверок для практики P09:

- `sbom.json` — SBOM проекта, сгенерированный Syft из содержимого репозитория на конкретном коммите.
- `sca_report.json` — отчёт SCA (Grype) по выявленным уязвимостям на основе `sbom.json`.
- `sca_summary.md` — текстовая сводка по уязвимостям (агрегация по severity, комментарии и план действий).

Файлы **не коммитятся вручную** — они генерируются в CI и доступны:

- как артефакты GitHub Actions (`sbom-sca-artifacts`),
- по пути `EVIDENCE/P09/…` в workspace конкретного job (можно отладочно просматривать в logs/steps).

Каждый запуск workflow привязан к конкретному коммиту, поэтому SBOM и SCA‑отчёт можно использовать
для **трассировки DS1** в итоговом отчёте (ссылка commit/job → артефакты → анализ).
